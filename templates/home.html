Мета проекту:

Мета проекту - навчитися створювати веб-сайти за допомогою фреймворку Django та розробляти функціональність для представлення особистого портфоліо.

Створення власного сайту з Django

Django - це онлайн-сервіс, який дозволяє вам створити своє власне професійне портфоліо в інтернеті. Ви можете показати свої навички, проекти та досягнення потенційним роботодавцям або клієнтам.

Основні можливості:

  • Створення Профілю: Додавайте свої дані про досвід роботи, освіту та навички.
  • Додавання Проектів: Публікуйте ваші проекти та роботи для представлення.
  • Зображення та Відео: Завантажуйте зображення та відео, щоб краще показати свої досягнення.
  • Перегляд Інших Профілів: Дивіться портфоліо інших користувачів для натхнення.
  • Пошук та Фільтрація: Шукайте інших фахівців та їхні роботи за різними критеріями.

Технології:

  • Фронтенд: HTML, CSS, JavaScript
  • Бекенд: Django, Python
  • База даних: SQLite
